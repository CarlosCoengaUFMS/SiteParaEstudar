Variável Significado Exemplo

int D = 4;                     // Dimensão da matriz

🧠 1. O que é uma Matriz Bidimensional?

🏢 Analogia: Prédio com Andares e Apartamentos

Se um vetor é um prédio com vários apartamentos enfileirados...

Uma matriz é como um prédio com VÁRIOS ANDARES, onde cada andar tem vários apartamentos.

  • L (Linha) = Andar do prédio
  • C (Coluna) = Apartamento naquele andar
  • matriz[L][C] = Valor guardado no andar L, apartamento C

Declaração de uma matriz:

int D = 3;
int matriz[D][D];   // D linhas × D colunas = D² elementos

float M[4][4];        // D=4: 4 linhas × 4 colunas = 16 elementos
int dados[6][6];      // D=6: 6 linhas × 6 colunas = 36 elementos

📝 Regra de ouro: O primeiro colchete é sempre a LINHA (L), o segundo é a COLUNA (C).

matriz[L][C]

📐 2. Entendendo as Diagonais (O CONCEITO MAIS IMPORTANTE!)

⚠️ Este é o coração da Lista 7! Se você entender as diagonais, resolve todos os exercícios.

📝 Notação: L = Linha, C = Coluna, D = Dimensão da matriz

A diagonal principal vai do canto superior esquerdo ao inferior direito.

Condição: L == C (Linha igual a Coluna)

⚠️ Atenção: Para a diagonal secundária e regiões relacionadas, use APENAS:

  • Estritamente acima: L + C < D - 1 (não inclui a diagonal)
  • Estritamente abaixo: L + C > D - 1 (não inclui a diagonal)
  • Na diagonal: L + C == D - 1 (apenas os elementos da diagonal)

NÃO use >= ou <= a menos que o exercício peça para incluir a diagonal na região acima/abaixo.

🔄 3. Como Percorrer uma Matriz com Laços

Leitura de todos os elementos:

int D = 4;  // Dimensão da matriz
int matriz[D][D];
int L, C;

// Laço externo: percorre as LINHAS
for(L = 0; L < D; L++) {
    // Laço interno: percorre as COLUNAS da linha atual
    for(C = 0; C < D; C++) {
        scanf("%d", &matriz[L][C]);
    }
}

📖 Analogia: Lendo um Livro

Percorrer uma matriz é como ler um livro:

  • O laço externo (L) vira as páginas (linhas)
  • O laço interno (C) lê cada palavra da página atual (colunas)

🔢 4. Exemplo Prático: Somatório da Diagonal Principal (Exercício 3)

Enunciado: Matriz 5x5 (D=5), somar elementos da diagonal principal.

1
Declare a matriz, D, L, C e a variável soma 
int D = 5;
int matriz[D][D];
int L, C;
int soma = 0;
2
Leia os D² valores (laço duplo com L e C) 
for(L = 0; L < D; L++)
    for(C = 0; C < D; C++)
        scanf("%d", &matriz[L][C]);
3
Some APENAS os elementos da diagonal principal (L == C) 
for(L = 0; L < D; L++) {
    soma += matriz[L][L];  // L == C, então usamos L nos dois índices!
}

💡 Perceba: não precisamos de dois laços! Apenas um for com matriz[L][L].

⭐ 6. Exemplo Prático: Números Perfeitos (Exercício 5)

Número perfeito: soma dos divisores (exceto ele mesmo) = número.

Ex: 6 → divisores: 1, 2, 3 → 1+2+3 = 6 ✅

1
Para cada número da matriz, calcule a soma dos divisores 
int D = 3;  // Matriz 3x3
int matriz[D][D];
int L, C, k;

for(L = 0; L < D; L++) {
    for(C = 0; C < D; C++) {
        int num = matriz[L][C];
        int soma_div = 0;
        
        // Laço para encontrar divisores (1 até num/2)
        for(k = 1; k <= num/2; k++) {
            if(num % k == 0) {
                soma_div += k;
            }
        }
        
        // Verifica se é perfeito
        if(soma_div == num) {
            printf("%d - Numero perfeito\n", num);
        } else {
            printf("%d - Nao e perfeito\n", num);
        }
    }
}

💡 Otimização: Só precisamos verificar divisores até num/2, pois nenhum divisor (exceto o próprio número) é maior que a metade.

🔍 7. Exemplo Prático: Produto das Diagonais (Exercício 8)

Enunciado: Matriz 4x4, calcular produto das duas diagonais e indicar o menor.

int D = 4;
float matriz[D][D];
float prod_princ = 1, prod_sec = 1;
int L, C;

// Leitura da matriz (omitido para focar no cálculo)

for(L = 0; L < D; L++) {
    for(C = 0; C < D; C++) {
        // Diagonal principal: L == C
        if(L == C) {
            prod_princ *= matriz[L][C];
        }
        // Diagonal secundária: L + C == D - 1
        if(L + C == D - 1) {
            prod_sec *= matriz[L][C];
        }
    }
}

printf("Produto diagonal principal: %.2f\n", prod_princ);
printf("Produto diagonal secundaria: %.2f\n", prod_sec);

// Comparação para achar o menor
if(prod_princ < prod_sec) {
    printf("Menor produto: %.2f - Diagonal principal\n", prod_princ);
} else if(prod_sec < prod_princ) {
    printf("Menor produto: %.2f - Diagonal secundaria\n", prod_sec);
} else {
    printf("Menor produto: %.2f - Ambas diagonais\n", prod_princ);
}

📋 8. Resumo: Checklist para Matrizes

  1. ✅ Defina D (dimensão da matriz quadrada)
  2. ✅ Declare a matriz: tipo nome[D][D]
  3. ✅ Declare L e C para os índices
  4. ✅ Leia com laço duplo: for(L) for(C) scanf
  5. ✅ Identifique a região de interesse:
    RegiãoCondição
    Diagonal PrincipalL == C
    Diagonal SecundáriaL + C == D - 1
    Acima da PrincipalC > L
    Abaixo da PrincipalC < L
    Acima da SecundáriaL + C < D - 1
    Abaixo da SecundáriaL + C > D - 1
  6. ✅ Calcule (soma, média, produto, etc.)
  7. ✅ Imprima o resultado
